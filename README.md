# CM4145_NLP_coursework
## Overview
This repository is for my 4th year Natural language processing coursework, where I will have a dataset called 'emotion_dataset' and using this dataset I will compare two significantly different NLP pipelines on a chosen dataset to determine which approach works better for a specific text-based task with google colab.
